# Student-registration

A simple web-based student registration system built with HTML, CSS, and JavaScript. This system allows users to register students, store their data in local storage, and view, edit, or delete registered student records.

## Features

-- Add new students
   Users can fill out a registration form with:
    -Name (letters only)
    -Student ID (numbers only)
    -Email address (email only)
    -Contact number (numbers only, max 10 digits)
-- Can view Registered Students
-- Can Edit Student record. (Except ID)
-- Can Delete Student record. 
-- Responsive design.
